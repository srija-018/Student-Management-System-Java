# Student Management System using Core Java

## Overview
A console-based Java application developed to manage student records efficiently using CRUD operations.

## Features
- Add Student
- View Students
- Search Student
- Delete Student

## Technologies Used
- Core Java
- ArrayList
- Object-Oriented Programming (OOP)

## Project Structure
- Student.java
- StudentManagement.java

## How to Run

1. Compile the files:
```bash
javac Student.java StudentManagement.java
```

2. Run the application:
```bash
java StudentManagement
```

## Concepts Used
- Classes & Objects
- Constructors
- Methods
- ArrayList
- Loops
- Switch Case
- Exception Handling
